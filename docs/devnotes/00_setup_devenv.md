# Setup environment for developing ice.

We use `poetry` as the virtualenv as well as project manager (e.g. dependencies, packaging, publishing, etc.).

```bash
git clone $REPO_ADDR ice-learn
cd ice-learn
pip install poetry
poetry install
```
